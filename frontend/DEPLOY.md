# Если на сайте не видно новых кнопок (проверка IP и т.д.)

Сайт **android-automation-frontend.onrender.com** показывает только то, что **собралось на Render** из ветки, привязанной к сервису (обычно `main`).

1. Закоммить изменения в **`frontend/`** (в корне репо).
2. **Push** в GitHub в ту ветку, откуда Render тянет код.
3. В [Render Dashboard](https://dashboard.render.com) → сервис **android-automation-frontend** → вкладка **Events** — дождись **Live** (зелёный деплой).
4. Открой форму создания задачи с **жёстким обновлением**: Ctrl+F5 (или инкогнито).

### Как понять, что подтянулась новая сборка

Под заголовком **«Создать задачу»** серая строка **`UI: proxy-ip-2026-03-22`**.  
Если её **нет** или другой текст — на Render всё ещё **старый** `dist`.

При следующих правках UI можно поменять метку в **`vite.config.ts`** → `__UI_BUILD_TAG__`.
